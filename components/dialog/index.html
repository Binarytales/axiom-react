Dialog

The Dialog component is an interface to the [react-modal](https://github.com/reactjs/react-modal) component. All properties you pass to Dialog will forward to the modal component.

  • React
<Dialog>
  <DialogHeader onRequestClose={function noRefCheck() {}}> ... </DialogHeader>
  <DialogBody> ... </DialogBody>
  <DialogFooter> ... </DialogFooter>
</Dialog>

Sized Dialogs

  • React
  • Raw HTML
<Dialog> ... </Dialog>
<Dialog size="medium"> ... </Dialog>
<Dialog size="large"> ... </Dialog>

  • React
<Dialog fullScreen={true}> ... </Dialog>
