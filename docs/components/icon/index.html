Icons

<Icon>

Show <Base> properties
inline
bool

Inline styling that allows the Icon to placed next to other inline elements or text.

name
Requiredenum

Name of the icon that will be displayed

size
stringDefault: 1rem

Size of icon (with unit)

spaceLeft
enum

Spacing applied to the left of the Icon. Must be used with the inline property

spaceRight
enum

Spacing applied to the right of the Icon. Must be used with the inline property

Component
unionDefault: div

The component that should be rendered with the Base component functionality. If a string is given then it must be a valid React registered element tag.

className
string

Class name to be appended to the element

hiddenUntil
enum

Control over when the element should be hidden until. Opposite of `visibleUntil`.

space
enum

Vertical margins given to the element

sticky
string

Distance from the top of the window where the element shall be stuck to. Utility for "position: sticky;".

textBreak
enum

Text wrap styling

textCase
enum

Text casing styling

textCenter
enum

Text central alignment either all of the time, with a value of `true` otherwise at one of the breakpoints specified.

textColor
enum

Text color styling

textEllipsis
bool

Text ellipsis styling

textEmphasize
bool

Text emphasis styling

textLeft
enum

Text left alignment either all of the time, with a value of `true` otherwise at one of the breakpoints specified.

textRight
enum

Text right alignment either all of the time, with a value of `true` otherwise at one of the breakpoints specified.

textSize
enum

Text size styling

textStrike
bool

Text strike styling

textStrong
bool

Text bold weight styling

textUnderline
bool

Text underlined styling

theme
enum

Theme styling modifier

visibleUntil
enum

Control over when the element should be visible until. Opposite of `hiddenUntil`.
