Position

<Position>

Show <Base> properties
children
Requiredarray

Children inside Position this should contain all of and only PositionContent and PositionTarget!

enabled
Default: truebool

Adds control to conditional enable or disabled the positioning logic. Must be true for isVisible to take effect.

flip
Default: clockwiseenum

Adds control of content flipping fallbacks.

isVisible
Requiredbool

Toggles the visibility of the PositionContent

offset
Default: middleenum

Controls the starting offset of the content

position
Default: topenum

Controls the starting position around PositionTarget in which the PositionContent will attempt to be placed. If that position is not available due to collision, it will be placed according to the flip behaviour until a valid position is found.

showArrow
Default: truebool

Toggle that allows the arrow of the Context component to be hidden

onMaskClick
func

When provided a mask will be placed behind PositionContent, where this function is called when clicked.

onPositionChange
func

Optional handler that is called, with the new position, when PositionContent has been positioned.

Component
Default: divunion

The component that should be rendered with the Base component functionality. If a string is given then it must be a valid React registered element tag.

className
string

Class name to be appended to the element

hiddenUntil
enum

Control over when the element should be hidden until. Opposite of `visibleUntil`.

space
enum

Vertical margins given to the element

sticky
string

Distance from the top of the window where the element shall be stuck to. Utility for "position: sticky;".

textBreak
enum

Text wrap styling

textCase

Text casing styling

textCenter
enum

Text central alignment either all of the time, with a value of `true` otherwise at one of the breakpoints specified.

textEmphasize

Text emphasis styling

textLeft
enum

textSize
