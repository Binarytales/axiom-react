Position

<Position>

Show <Base> properties
children
Requiredarray

Children inside Position this should contain all of and only PositionContent and PositionTarget!

enabled
bool

Adds control to conditional enable or disabled the positioning logic. Must be true for isVisible to take effect.

true
isVisible
Requiredbool

Toggles the visibility of the PositionContent

true
offset
enum

Controls the starting offset of the content

parentNode
object

An HTML element that holds the dynamically inserted PositionContent children. By default this will be inserted into an element on the body.

position
enum

Controls the starting position around PositionTarget in which the PositionContent will attempt to be placed. If that position is not available due to collision, it will be placed clockwise from that position until a valid position is found.

onMaskClick
func

When provided a mask will be placed behind PositionContent, where this function is called when clicked.
