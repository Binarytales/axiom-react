Dot Plot

Basic DotPlotChart

DotPlotChart automatically stacks dots which have the same value.

DotPlotChart expects a data prop in this format:

[{
  label: 'Label A',
  values: {
    blue: 0,
    pink: 98,
    lilac: 45
  },
}, {
  label: 'Label B',
  values: {
    blue: 90,
    pink: 10,
    lilac: 55
  },
}, {
  label: 'Label C',
  values: {
    blue: 45,
    pink: 80,
    lilac: 20
  },
}]

And a chartKey prop in this format:

[{
    color: 'blue',
    label: 'Data Set A'
  },
  {
    color: 'pink',
    label: 'Data Set B'
  },
  {
    color: 'lilac',
    label: 'Data Set C'
  },
]
