<ColumnChart>

Show <Base> properties
ContextComponent
func

Contextual component that appears when clicking on a bar. It is provided with the `color`, `label`, and `value` that has been clicked on, along with the `data` that was given for that row.

chartKey
RequiredarrayOf

The key that is shown along the bottom of the axis. It is also used to determine the order of bars.

data
RequiredarrayOf

The data used to render the bars along the top. The `label` is used for the xAxis and `values` for bars

direction
enum

Direction the bars point

height
Requiredstring

Overall height of the individual column charts

label
node

Label to show on the top yAxis row

labelColumnWidth
stringDefault: 0rem

The width of the yAxis labels columns

showBarLabel
bool

Control for showing the percentage labels above the Bars

showSubLabel
bool

Control for showing the labels below the Bars

showXAxisLabels
bool

Control for showing the xAxis labels
